> ## 技术栈
>
> `uni-app` `vue2` `vuex` `node`  
>
> ## 项目结构
>
> ```
> 
> .
> 
> ├── .hbuilderx // 云函数配置文件
> 
> ├── node_modules // 依赖的第三方包
> 
> ├── src // 源代码
> 
> │      ├── components // 通用组件
> 
> │      ├── pages // 主页面
> 
> │      ├── static // 静态资源
> 
> │      ├── store // 状态管理
> 
> │      ├── util // 公用方法
> 
> │      └── main.js 项目入口文件
> │      └── pages.json uniapp路由配置
> │      └── manifest.json uniapp打包配置
> │      └── uni.css 主题配置文件
> 
> ├── .babelrc // babel 配置
> 
> ├── package.json // 项目信息文件
> 
> └── README.md // 说明
> 
> .
> 
> ```
>

> ## 项目启动
>
> ```bash
> 
> # 克隆到本地 
> $ git clone https://gitee.com/ljwzuishuai/cloud_music.git
> 
> # cd 到项目目录
> $ cd cloud_music
> 
> # 安装依赖
> $ npm install
> $ yarn install (推荐使用)
> 
> # 启动开发环境
> $ npm run serve
> $ yarn serve
> 
> ```
